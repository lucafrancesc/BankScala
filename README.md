# Bank Tech Test in Scala

### User Stories:
```
As a user,
So that I can manage my money,
I want to be able to create a bank account.

As a user,
So that I can use my account,
I want to be able to deposit money into my account.

As a user,
So that I can use my account,
I want to be able to withdraw money from my account.

As a user,
So that I can keep track of my transactions,
I want to be able to print a statement in the following format:

Date || Credit || Debit || Balance
2019-04-21 || || 500.00 || 2500.00
2019-04-20 || 2000.00 || || 3000.00
2019-04-20 || 1000.00 || || 1000.00
```
